Capturing a screenshot in org-mode

Org-mode in Emacs is one of best writing tool for me.

When I memorized something, I used to open an another tool such as Evernote, MS Onenote or something. These tools have WYSIWYG interface, which seem to be quite user-friendly.

What I stopped using the tools is it forced me to think about layouts. I wanted to think about only contents but visual.

I started use org-mode, it goes well. I can focus only writing.

But it annoys me when I paste images.

So I wrote change below:

init.el is like this:

(require 'org-attach-screenshot)
(setq org-attach-screenshot-command-line "screencapture -s %f")

I want to let the big image thumbnailed and linking to originai image. I change original .el file like this.

-          (insert (concat "[[file:" linkfilename "]]"))
+          (insert (concat "#+attr_html: :width 300px\n[[file:" linkfilename "][file:" linkfilename "]]"))
